## Deployment

The site is automatically deployed to [cuongw.surge.sh](https://cuongw.surge.sh) when changes are pushed to the main branch.

### Setup Deployment

To enable automatic deployment, add these secrets to your GitHub repository:

1. `SURGE_LOGIN` - Your Surge.sh email
2. `SURGE_TOKEN` - Your Surge.sh token (get it by running `surge token`)

## Development

Simply open `index.html` in your browser or use a local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`
